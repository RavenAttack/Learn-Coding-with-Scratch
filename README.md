# Learn-Coding-with-Scratch

CodeBlocks Lab is a Scratch-inspired learning playground where each visual block
maps to a real code line in **Python**, **C++**, **Lua**, or **Rust**.

## What it does

- Keeps the familiar `when green flag clicked` flow from Scratch.
- Lets learners stack simple blocks to build game logic quickly.
- Shows generated code in multiple programming languages from the same block
  script.
- Allows direct editing of each code line inside the block to teach syntax.
- Supports custom blocks with your own language lines and runtime behavior.
- Includes a `WasmRunner` hook so you can plug in real WebAssembly runtimes
  later.

## Run locally

Open `index.html` in a browser.

## Project files

- `index.html` – layout and panels
- `styles.css` – Scratch-like visual styling
- `app.js` – block model, language mapping, runtime, and custom blocks
